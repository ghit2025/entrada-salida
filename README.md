La herramienta que se utiliza en la realización de esta práctica es el simulador del MC68000 bsvc. Este simulador incluye el programa ensamblador 68kasm y el fichero de configuración del computador emulado. Bsvc se ejecuta en gran variedad de plataformas entre ellas Linux. 
## Proyecto

### Entrada/Salida mediante interrupciones
El alumno deberá programar una operación completa de E/S, mediante interrupciones vectorizadas, sobre un procesador (MC68000) simulado mediante la herramienta de libre distribución que se describe y utiliza en la práctica optativa de E/S (bsvc). Para ello, deberá inicializar el módulo de entrada/salida y la tabla de vectores de interrupción, realizar la rutina de servicio de interrupciones y las rutinas de inicialización y finalización de la operación. 

## Práctica de Entrada/Salida

El objetivo es la toma de contacto con la herramienta que se utilizará para la realización del Proyecto de la Asignatura. En esta práctica se propone la implementación de un manejador de un dispositivo sencillo (línea serie) mediante Entrada/Salida programada. Para ello se utilizará una herramienta de libre distribución (bsvc) para la simulación de procesadores y otros dispositivos.
